# The Cephas Church International Website

A modern, responsive church website built with **HTML5, CSS3, and JavaScript**, designed to showcase the vision, leadership, programs, service schedules, and community activities of **The Cephas Church International (TCCI Global).**

This project focuses on combining **clean UI design, accessibility, responsiveness, and interactive user experience** into a real-world organizational website.

---

## Project Overview

The Cephas Church International website was created to provide visitors and members with easy access to:

* Church vision, mission, and core values
* Service schedules and locations
* Annual church programs
* Leadership profiles
* Service unit opportunities
* Testimonials and media content
* Prayer request submission

---

## Features

### Responsive Design

* Fully responsive across desktop, tablet, and mobile devices
* Flexible layouts using CSS Grid and Flexbox

### Interactive Navigation

* Smooth scrolling navigation
* Hover animations
* User-friendly section linking

### Leadership Showcase

* Leadership profile cards
* Image display with responsive layouts

### Multimedia Integration

* Embedded YouTube sermon/messages
* Testimonial image slideshow

### Form Validation

Client-side JavaScript validation for:

* First name
* Last name
* Email address
* Prayer request input

Validation includes:

* Required field checks
* Character length checks
* Email pattern validation
* Invalid character detection
* Real-time textarea feedback

### Automatic Slideshow

Testimonials section includes:

* Auto-slide every 3 seconds
* Previous/Next controls
